Python Program: Even or Odd Checker

Code (User Input)

num = int(input("Enter a number: ")) if num % 2 == 0: print("The number is Even.") else: print("The number is Odd.")

Input

Enter a number: 13

Output

The number is Odd.

Code (Assigned Value)

num = 8 if num % 2 == 0: print("The number is Even.") else: print("The number is Odd.")

Input

Value Assigned: 8

Output

The number is Even.
